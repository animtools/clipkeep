# ClipKeep

> 動画の"あの瞬間"を、二度と見失わない。

ClipKeep は動画をシーン単位のタイルに分解し、お気に入りの瞬間を Keep してコレクションできる、Windows 向けのローカル動画ライブラリアプリです。

本リポジトリ（[`animtools/clipkeep`](https://github.com/animtools/clipkeep)）は **エンドユーザー向けの配布面**です。ソースコードは含みません。アプリ本体は [Releases](https://github.com/animtools/clipkeep/releases) から入手してください。

## ダウンロード

[Releases](https://github.com/animtools/clipkeep/releases) から最新のポータブル ZIP（またはインストーラ）を取得してください。

- 動作環境: Windows 10 / 11（64bit）。WebView2 ランタイム（Windows 11 は標準搭載のことが多い）
- 初回起動時に SmartScreen の警告が出ることがあります（[QUICKSTART.md](./QUICKSTART.md) 参照）
