# SUPPORT — ClipKeep

不具合報告・質問・要望をお待ちしています。

---

## 連絡方法

- **不具合・要望**: [GitHub Issues](https://github.com/animtools/clipkeep/issues)

## 不具合報告のときに書いていただきたいこと

1. **現象** — 期待した動作と、実際の動作  
2. **再現手順** — 1. … → 2. … → 3. …  
3. **環境** — Windows のバージョン、ClipKeep のバージョン  
4. **スクリーンショット** — あれば添付  
5. **ライブラリ規模** — おおよその動画本数・Keep 枚数（性能問題の切り分け用）

## よくある質問（抜粋）

- **データはどこ？** — 端末内（ライブラリ DB・タイル画像）。クラウドへ自動送信しません。  
- **無料枠を超えたら？** — アプリのライセンス管理からアップグレードできます。既存データは消えません。  
- **サブスクを止めたい** — [カスタマーポータル](https://polar.sh/shinosuke/portal)（購入時メールのリンクからも可）
