# QUICKSTART — ClipKeep

最短で「ダウンロード → 起動 → 動画を入れて Keep する」まで進める手順です。詳細は [USER_GUIDE.md](./USER_GUIDE.md) を参照してください。

---

## 1. ダウンロードと起動

1. [Releases](https://github.com/animtools/clipkeep/releases) から最新の配布 ZIP を取得する  
2. 任意のフォルダに解凍する  
3. `ClipKeep.exe`（または配布物の案内どおりの exe）を起動する  

未署名のため、Windows の SmartScreen で警告が出ることがあります。「詳細情報」→「実行」で進められます。

## 2. 最初の動画

1. ライブラリに動画ファイルまたはフォルダを追加する  
2. タイル生成が終わるまで待つ（バックグラウンドで進みます）  
3. 好きなシーンのタイルを開き、Keep する  

## 3. つまずいたら

- [SUPPORT.md](./SUPPORT.md) の報告項目を添えて Issues へ
